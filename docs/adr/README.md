# Architecture Decision Records (ADRs)

ADRs capture significant technical decisions with context and consequences. Use date- and number-prefixed filenames.

Template sections:
- Title
- Date
- Status (proposed | accepted | superseded | deprecated)
- Context
- Decision
- Consequences
- Links (Supersedes / Superseded by / Related)
